
# EtyDict
EtyDict is a free online dictionary and thesaurus explorer.

# Features
 - Advanced search
 - Rich database
 - Dynamic etymology graph
 - Account and profile
 - Word favorites and history
 - Popular and trending words
 - User browsing data for further analytics 

# Requirements

 - Composer
 - Twig
 - aws/aws-sdk-php
 - XAMPP

For IPA pronunciation, AWS tokens are required.


# Installation

1. Extract the folder into php/htdocs
2. install composer requirements
3. NOTE: expose ./public/
